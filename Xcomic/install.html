Installation Instructions

System Requirements

Please make sure that the system you plan to install Xcomic has the following:
  • PHP version 4.0 or higher.
  • MySQL version 3.0.0 or higher. Currently, Xcomic does not support other databases.

Steps to Install

  1. Unzip the package in an empty directory (ie. /xcomic)
  2. chmod 777 includes and chmod 777 comics.
  3. Launch /admin/install/ in your browser. Enter in your database information and the URL paths to your Xcomic installation.
  4. Remember what password you use for your administrator user.
  5. The install script should then send you to the login page. Sign in with the username and password you chose during the installation. You can then click on 'Users' to change the password.
  6. Finally, chmod 755 includes. Congratulations, if everything went well, you have successfully installed Xcomic!

Troubleshooting

If you have trouble or errors installing Xcomic, check the following:
  • Make sure the directories includes and comics are chmod 777 (that means world writable).
  • Make sure you have the correct database information. Only MySQL is officially supported in version 0.9.0
  • Make sure you entered the URL to Xcomic correctly.
  • Still having problems? Get Support and Report the bug

Upgrading

Important! Always make a backup of your files AND your database before upgrade, in case something goes wrong! Contact your web hosting provider for information on creating a backup if you do not know how.

If you are running version 0.8.x and would like to upgrade to version 0.9.0, replace all of the old files with the files from the new version. You must then go to admin/install/upgrade.php to finish the upgrade. The upgrade script will finalize the upgrade process.

If you are (gasp) running a version of Xcomic before 0.8.0, please perform a full install of version 0.9.0 and manually port over your comic entries.
