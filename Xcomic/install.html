Installation Instructions

System Requirements

Please make sure that the system you plan to install Xcomic has the following:
  • PHP version 4.0 or higher.
  • MySQL version 3.0.0 or higher. Currently, Xcomic does not support other databases.

Steps to Install

  1. Unzip the package in an empty directory (ie. /xcomic)
  2. chmod 777 includes and chmod 777 comics.
  3. Launch /admin/install.php in your browser. Enter in your database information and the URL paths to your Xcomic installation.
  4. Note the password given to you.
  5. The install script should then send you to the login page. Sign in with the username admin and the password given to you during the installation. You can then click on 'Users' to change the password.
  6. Finally, chmod 755 includes. Congratulations, if everything went well, you have successfully installed Xcomic!

Troubleshooting

If you have trouble or errors installing Xcomic, check the following:
  • Make sure the directories includes and comics are chmod 777 (that means world writable).
  • Make sure you have the correct database information. Only MySQL is supported in version 0.8.5
  • Make sure you entered the URL to Xcomic correctly.
  • Still having problems? Report the bug

Upgrading

If you are running version 0.8.0 and would like to upgrade to version 0.8.5, just replace all of the old files with the files from the new version.

If you are (gasp) running a version of Xcomic before 0.8.0, please perform a full install of version 0.8.5 and manually port over your comic entries. Would someone be willing to write an update.php script for this?
